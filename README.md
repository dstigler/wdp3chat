Installationsanleitung:
1.) MongoDB installieren
2.) "npm install" ausführen
3.) im config.js-File falls notwendig Port ändern
4.) MongoDB starten "sudo service mongodb start"
5.) "node server.js" ausführen

Wdp3chat ist ein Gruppenchat, auf Basis von NodeJS, Bootstrap, MongoDB.
Funktionen: Signup, Login, Chaten, Räume erstellen.
